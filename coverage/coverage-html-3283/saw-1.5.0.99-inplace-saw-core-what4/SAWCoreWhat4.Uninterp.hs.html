    1 {- |
    2 Given a constant of type `T`, construct an uninterpreted
    3 constant with that type.  The type `T` is *not* limited to What4 BaseTypes or
    4 FirstOrderTypes, and we support polymorphic and function types.
    5 Note that this is used for both uninterpreted functions and when we create
    6 symbolic variables.
    7 
    8 The Algorithm for Uninterpreted Functions
    9 =========================================
   10 
   11 Our strategy is to generate a separate uninterpreted function for each
   12 base type that is mentioned in the result of the original uninterpreted
   13 function.  If we need more than one value of a type, then we return an
   14 *array* of values, and the actual results of the original function are
   15 obtained by selecting elements of this array.
   16 
   17 For example, consider a function `f: [16] -> ([4][8],Bool)`.
   18 A call `f x` will be translated like this:
   19 
   20   f_bv8:  [16] -> Array [2] [8]   // Uninterpreted
   21   f_bool: [16] -> Bool            // Uninterpreted
   22   w8s   = f_bv8 x                 // This has some auto-generated name
   23   bools = f_bool x                // This has some auto-generated name
   24 
   25 Value for `f x`:
   26   ([ w8s ! 0, w8s ! 1, w8s ! 2, w8s ! 3 ], bools)
   27 
   28 The reason we do this is to reduce the size of terms in examples like this:
   29 
   30   g: [1024][8] -> [1024][8]
   31 
   32 `g` becomes a function of 1024 [8] arguments.  With the naive translation
   33 we'd end up with:
   34 
   35   g_0 a0 .. a1023
   36   g_1 a0 .. a1023
   37   ..
   38   g_1023 a0 .. a1023  
   39 
   40 Note that this has 1024 * 1024 terms.  With the array translation, we
   41 should end up with:
   42 
   43   arr = g_u8 a0 .. a0123
   44   el0 = arr @ 0
   45   el1 = arr @ 1
   46   ..
   47   el1023 = arr @ 1023
   48 
   49 This is of size 1024 + 1024, which is much smaller.
   50 
   51 
   52 Handling Functions and Polymorphism (UnintApp, SymFnCache)
   53 ==========================================================
   54 
   55 When working with a function type (`Pi`), we don't generate uninterpreted
   56 functions immediately.  Instead, we generate a Haskell function that collects
   57 the arguments and only generates the uninterpreted function when they are
   58 all available (see `VFun`) (i.e., the first time the uninterpreted function
   59 is called).   Once the functions are generated they are stored in a cache
   60 (see `SymFnCache`), so that future calls to the same function do not generate
   61 new uninterpreted symbols, but reuse the old ones.  This is not an optimization,
   62 but is crucial for the correct behavior of the algorithm.
   63 
   64 The `UnintApp` type is used to collect the arguments to a function, and also
   65 compute the root name for the function.   For ordinary (non-dependent) 
   66 function applications we just collect the arguments in the `UnintApp`.  However,
   67 for dependent applications (e.g., to handle a size polymorphic function),
   68 we instead modify the root name of the function.  This means that different
   69 size instantiations of the functions end up with different uninterpreted
   70 function instances.   Here's an example:
   71 
   72 sum : {n} (fin n) => [n][8] -> [8]
   73 xs: [16][8]
   74 ys: [32][8]
   75 
   76 `sum xs` will result in something like this:
   77 
   78 sum_16_bv8: [8] -> [8] ..  [8] -> [8]     /* Has 16 arguments */
   79 sum_16_bv8 (xs @ 0) (xs @ 1) .. (xs @ 15)
   80 
   81 otoh, `sum ys` will result in something like this:
   82 sum_32_bv8: [8] -> [8] ..  [8] -> [8]     /* Has 32 arguments */
   83 sum_32_bv8 (ys @ 0) (ys @ 1) .. (ys @ 31)
   84 
   85 See `applyUnintApp` for the full details.
   86 
   87 
   88 Reinterpreting Terms Back Back to SAW Core (ArgTerm)
   89 ====================================================
   90 
   91 Also we have two cases to consider:
   92   1. When translating to the solver (`parseUninterpreted`)
   93   2. When doing symbolic simulation for SAW (`parseUninterpretedSAW`).
   94 
   95 The difference between the two is that in case (2) we need to also be able
   96 to go back from the symbolic expressions to a semantically equivalent SAW
   97 Core term.  To do this, we need to provide interpretations for all the functions
   98 that were left as uninterpreted during the original translation.  For example,
   99 we'd need to register the following interpretations for the examples above:
  100 
  101 f_bv8 x = [ tup1 @ 0, tup1 @ 1, tup1 @ 2, tup1 @ 3 ]
  102   where tup1 = (f x).0
  103 
  104 f_bool x = (f x).1
  105 
  106 g_u8 x0 x1 .. x1023 = [ fx @ 0, fx @ f1, ... fx @ 1023 ]
  107   where fx = f [ x0, x1 .., x1023 ]
  108 
  109 Note 1: the sharing in this examples (the `where` clause) happens
  110 automatically due to sharing inherent in SAW Core terms.
  111 
  112 Note 2: we also need to reconstruct the arguments to the original functions
  113 from their components before calling (e.g., as in `g_u8`).
  114 
  115 We use the type `ArgTerm` to help us create the interpretation.  The
  116 constructors in `ArgTerm` are used to reassemble the arguments to the original
  117 function out of their pieces (e.g., put together the 1024 parameters into
  118 a single vector), while the selectors are used to connect a part of the
  119 result of the original function, to the relevant uninterpreted bit.
  120 
  121 Note 3: Since this code is used for making symbolic variables as well as
  122 uninterpreted functions, there's a special case for terms that started as
  123 SAW Core *variables* rather than SAW Core *constant*, and don't have any
  124 arguments---these use `bindSAWTerm` instead of `sawRegisterSymFunInterp`.
  125 -}

  188 -- | Uninterpreted function results for a particular base type.
  189 -- The length of the lists should match the @UnintCount tc@ for the type.
  190 data Arr sym tc = forall args res. Arr {
  191   arrFn :: W.SymFn sym args res,
  192   -- ^ The uninterpreted function name.
  193 
  194   arrIndexWidth :: !Natural,
  195   -- ^ How many bits to use for the array index.
  196 
  197   arrSymExprs  :: [SymExpr sym tc],
  198   -- ^ Available symbolic expressions
  199 
  200   arrFromSym   :: [ArgTerm]
  201   -- ^ How to map `n`-th symbolic expression back into a SAW Core term.
  202   -- Note that these are in reverse order---every time we take a thing
  203   -- from `arrSymExprs`, we cons a new thing onto here.
  204 }

  206 -- | Indicates if we need to map What4 terms back to SAW.
  207 data ReturnTrip sym =
  208     NoReturnTrip sym
  209     -- ^ No need to reinterpret terms back into SAW Core.
  210 
  211   | forall n st fs. (sym ~ B.ExprBuilder n st fs ) =>
  212     DoReturnTrip sym Bool (SAWCoreState n) SharedContext ArgTerm
  213     -- ^ We should reinterpret terms back into SAW Core.
  214     -- The boolean flag indicates that this is a symbolic variable
  215     -- (instead of constant), which has special handling without arguments.
  216   
  217 withSym :: IsSymExprBuilder sym => ReturnTrip sym -> (sym -> a) -> a
  218 withSym xs k =
  219   case xs of
  220     NoReturnTrip sym -> k sym
  221     DoReturnTrip sym _ _ _ _ -> k sym
  222 
  223 mapArgTerm :: (ArgTerm -> ArgTerm) -> ReturnTrip sym -> ReturnTrip sym
  224 mapArgTerm f saw =
  225   case saw of
  226     NoReturnTrip sym -> NoReturnTrip sym
  227     DoReturnTrip sym isVar st sc t -> DoReturnTrip sym isVar st sc (f t)

  231 -- | Generate a call to an uninterpreted function, without reinterpreting
  232 -- back into SAW Core.
  233 parseUninterpreted ::
  234   forall sym.
  235   (IsSymExprBuilder sym) =>
  236   sym                         {- ^ How to make symbolic expressions -} ->
  237   IORef (SymFnCache sym)      {- ^ Cache of known uninterpreted functions -} ->
  238   UnintApp (SymExpr sym)      {- ^ Name of function and its arguments -} ->
  239   TValue (What4 sym)          {- ^ Type of the function's result -} ->
  240   IO (SValue sym)             {- ^ The symbolic value for the call to the uninterpreted function -}
  241 parseUninterpreted sym a b c = parseUninterpretedTop (NoReturnTrip sym) a b c
  242 
  243 -- | Generate a call to an uninterpreted function, and also register mapping
  244 -- back into SAW Core.
  245 parseUninterpretedSAW ::
  246   forall n st fs.
  247   B.ExprBuilder n st fs     {- ^ How to make symbolic expressions -} ->
  248   SAWCoreState n            {- ^ Register interpretations for terms here -} ->
  249   SharedContext             {- ^ Use this to build SAW Core terms -} ->
  250   IORef (SymFnCache (B.ExprBuilder n st fs)) {- ^ Cache of known uninterpreted functions -} ->
  251   Bool                      {- ^ Is the original term a variable (True), or constant (False) -} ->
  252   Term                      {- ^ Original term -} ->
  253   UnintApp (SymExpr (B.ExprBuilder n st fs)) {- ^ Type of the function's result -} ->
  254   TValue (What4 (B.ExprBuilder n st fs)) {- ^ Type of the function's result. -} ->
  255   IO (SValue (B.ExprBuilder n st fs))
  256 parseUninterpretedSAW sym st sc ref isVar term app t =
  257   parseUninterpretedTop (DoReturnTrip sym isVar st sc (ArgTermConst term)) ref app t

  259 parseUninterpretedTop ::
  260   forall sym.
  261   (IsSymExprBuilder sym) =>
  262   ReturnTrip sym ->
  263   IORef (SymFnCache sym)      {- ^ Cache of known uninterpreted functions -} ->
  264   UnintApp (SymExpr sym)      {- ^ Name of function and its arguments -} ->
  265   TValue (What4 sym)          {- ^ Type of the function's result -} ->
  266   IO (SValue sym)             {- ^ The symbolic value for the call to the uninterpreted function -}
  267 
  268 -- This case deals with symbolic variable with no parameters.  The round
  269 -- trip for these works differently (see `mkUninterpreted`),
  270 -- and since they have no arguments, we don't need to do the array caching.
  271 parseUninterpretedTop rt@(DoReturnTrip _ True _ _ _) ref app@(UnintApp _ _ argTys) ty =
  272   case testEquality Ctx.empty argTys of
  273     Just Refl -> evalStateT (parseUninterpreted' rt ref app ty) MapF.empty
  274     Nothing   -> fail "At present, we do not support symbolic variables with parameters"
  275     
  276 parseUninterpretedTop saw ref app ty =
  277   do
  278     count <-
  279       withSym saw (\sym ->
  280       MapF.traverseWithKey (mkUnint sym ref app) (countUninterpreted 1 MapF.empty ty))
  281     (val,st) <- runStateT (parseUninterpreted' saw ref app ty) count
  282     case saw of
  283       NoReturnTrip _ -> pure ()
  284       DoReturnTrip sym _ sawSt sc _ -> MapF.traverseWithKey_ register st
  285         where
  286         register :: BaseTypeRepr t -> Arr sym t -> IO ()
  287         register k (Arr fn ixW _ vs) =
  288           do
  289             elTy <- baseSCType sym sc k
  290             sawRegisterSymFunInterp sawSt fn $ \sc' ts ->
  291               do
  292                 terms <- mapM (\ate -> reconstructArgTerm ate sc' ts) (reverse vs)
  293                 case terms of
  294                   [t] -> pure (UninterpOne t)
  295                   _   -> pure (UninterpMany ixW elTy (V.fromList terms))
  296 
  297     pure val

  300 -- | Track how many uninterpreted results we need for each base type.
  301 newtype UnintCount (tc :: BaseType) = UnintCount Natural 
  302 
  303 -- | Count how many uninterpreted symbols we need to represent a value
  304 -- of the given type.  Note that this function should match exactly what
  305 -- is needed by `parseUninterpreted'`.
  306 countUninterpreted ::
  307   IsSymExprBuilder sym =>
  308   Natural -> MapF BaseTypeRepr UnintCount -> TValue (What4 sym) -> MapF BaseTypeRepr UnintCount
  309 countUninterpreted scale count ty =
  310   case ty of
  311     VPiType {}      -> count
  312     VBoolType       -> add BaseBoolRepr count
  313     VIntType        -> add BaseIntegerRepr count
  314     VIntModType {}  -> add BaseIntegerRepr count
  315     VRationalType   -> add BaseIntegerRepr (add BaseIntegerRepr count)
  316     VVecType n VBoolType ->
  317       case somePosNat n of
  318         Just (Some (PosNat w)) -> add (BaseBVRepr w) count
  319         _                      -> count
  320 
  321     VVecType n et ->
  322       if n == 0 then count else countUninterpreted (n * scale) count et
  323 
  324     VArrayType ity ety
  325       | Just (Some idx_repr) <- valueAsBaseType ity
  326       , Just (Some elm_repr) <- valueAsBaseType ety
  327       -> add (BaseArrayRepr (Ctx.Empty Ctx.:> idx_repr) elm_repr) count
  328 
  329     VDataType (ModuleIdentifier "Prelude.UnitType") [] [] -> count
  330 
  331     VDataType (ModuleIdentifier "Prelude.PairType") [TValue ty1, TValue ty2] [] ->
  332       countUninterpreted scale (countUninterpreted scale count ty1) ty2
  333 
  334     VDataType (ModuleIdentifier "Prelude.RecordType")
  335       [VString _fname, TValue ty1, TValue ty2] [] ->
  336       countUninterpreted scale (countUninterpreted scale count ty1) ty2
  337 
  338     _ -> count
  339   where
  340   add ::
  341     BaseTypeRepr tc ->
  342     MapF BaseTypeRepr UnintCount ->
  343     MapF BaseTypeRepr UnintCount
  344   add bt =
  345     MapF.insertWith
  346       (\(UnintCount x) (UnintCount y) -> UnintCount (x + y))
  347       bt (UnintCount scale)

  353 -- | Generate the uninterpreted functions.
  354 mkUnint ::
  355   forall sym tc.
  356   (IsSymExprBuilder sym) =>
  357   sym                         {- ^ How to make symbolic expressions -} ->
  358   IORef (SymFnCache sym)      {- ^ Cache of known uninterpreted functions -} ->
  359   UnintApp (SymExpr sym)      {- ^ Name of function and its arguments -} ->
  360   BaseTypeRepr tc             {- ^ Type of value -} ->
  361   UnintCount tc               {- ^ How many values we need -} ->
  362   IO (Arr sym tc)
  363 mkUnint sym ref (UnintApp nm args tys) ret (UnintCount tot)
  364   | let lim = tot - 1
  365   , let wn = width lim
  366   , Just (Some w) <- someNat wn =
  367     case isZeroOrGT1 w of
  368 
  369       -- We need only a single symbolic value (i.e., 0-bit index)
  370       Left Refl ->
  371         do
  372           fn <- mkSymFn sym ref (nm ++ suff ret) tys ret
  373           el <- W.applySymFn sym fn args
  374           pure (Arr { arrFn = fn, arrIndexWidth = wn, arrSymExprs = [el], arrFromSym = [] })
  375 
  376       -- We need multiple symbolic values of this type, result is array
  377       Right LeqProof ->
  378         do
  379           fn <- mkSymFn sym ref (nm ++ suff ret) tys (BaseArrayRepr (Ctx.singleton (BaseBVRepr w)) ret)
  380           arr <- W.applySymFn sym fn args
  381           els <- forM [ 0 .. lim ] $ \i ->
  382             do
  383               ind <- W.bvLit sym w (BV.mkBV w (fromIntegral i))
  384               W.arrayLookup sym arr (Ctx.singleton ind)
  385           pure (Arr { arrFn = fn, arrIndexWidth = wn, arrSymExprs = els, arrFromSym = [] })
  386 
  387   | otherwise =
  388     panic "parseUninterpreted" ["`someNat` returned `Nothing` for `Natural`"]
  389 
  390   where
  391   -- | Compute the number of bits required to represent the given integer,
  392   -- used to index the array of uninterpreted values.
  393   width :: Natural -> Natural
  394   width x = go 0 x
  395     where
  396     go s 0 = s
  397     go s n = let s' = s + 1 in s' `seq` go s' (n `shiftR` 1)
  398 
  399   -- Type suffixes for uninterpreted functions of each base type.
  400   suff :: BaseTypeRepr a -> String
  401   suff t =
  402     case t of
  403       BaseBoolRepr -> "_bool"
  404       BaseIntegerRepr -> "_int"
  405       BaseRealRepr -> "_real"
  406       BaseComplexRepr -> "_complex"
  407       BaseBVRepr w -> "_bv" ++ show w
  408       BaseFloatRepr (FloatingPointPrecisionRepr x y) -> "_float_" ++ show x ++ "_" ++ show y
  409       BaseStringRepr w -> "_str" ++ strw
  410         where strw = case w of
  411                        Char8Repr -> "8"
  412                        Char16Repr -> "16"
  413                        UnicodeRepr -> "U"
  414       BaseArrayRepr i e -> "_fun" ++ suff_asgn i ++ "_to" ++ suff e
  415       BaseStructRepr flds -> "_struct" ++ suff_asgn flds ++ "_end"
  416 
  417   suff_asgn i = intercalate "_and" (V.toList (Ctx.toVector i suff))
  418

  421 parseUninterpreted' ::
  422   forall sym.
  423   (IsSymExprBuilder sym) =>
  424   ReturnTrip sym ->
  425   IORef (SymFnCache sym) ->
  426   UnintApp (SymExpr sym) ->
  427   TValue (What4 sym) ->
  428   StateT (UnintState sym) IO (SValue sym)
  429 parseUninterpreted' saw ref app ty =
  430   case ty of
  431     VPiType t1 body ->
  432       pure $ VFun $ \x ->
  433         do
  434           x'   <- force x
  435           app' <- withSym saw (\sym -> applyUnintApp sym app x')
  436           t2   <- applyPiBody body (ready x')
  437 
  438           saw' <-
  439             case saw of
  440               NoReturnTrip sym -> pure (NoReturnTrip sym)
  441               DoReturnTrip sym isVar st sc trm ->
  442                 do newArg <- mkArgTerm sc t1 x'
  443                    let newTerm = ArgTermApply trm newArg
  444                    pure (DoReturnTrip sym isVar st sc newTerm)
  445 
  446           parseUninterpretedTop saw' ref app' t2
  447 
  448     VBoolType
  449       -> VBool <$> mkUninterpreted BaseBoolRepr saw
  450 
  451     VIntType
  452       -> VInt <$> mkUninterpreted BaseIntegerRepr saw
  453 
  454     VIntModType n
  455       -> VIntMod n <$> mkUninterpreted BaseIntegerRepr (mapArgTerm (ArgTermFromIntMod n) saw)
  456 
  457     VRationalType ->
  458       do
  459         -- See:
  460         -- https://github.com/GaloisInc/saw-script/issues/3206
  461         -- https://github.com/GaloisInc/what4/issues/364 
  462         -- Note that the `bad` would only matter if we use a rational in the
  463         -- result of an uninterpreted function, and we need to import the
  464         -- resulting What4 term back into What4
  465         let bad = ArgTermErr ("Rationals in the results of uninterpreted functions are unsupported")
  466         numer <- mkUninterpreted BaseIntegerRepr (mapArgTerm (\_ -> bad) saw)
  467         -- TODO(#2433): Assert that the denominator is non-zero.
  468         denom <- mkUninterpreted BaseIntegerRepr (mapArgTerm (\_ -> bad) saw)
  469         pure $ VRational numer denom
  470 
  471     VVecType n VBoolType ->
  472       case somePosNat n of
  473         Just (Some (PosNat w)) -> VWord . DBV <$> mkUninterpreted (BaseBVRepr w) saw
  474         _                      -> pure (VWord ZBV)
  475 
  476     VVecType n et ->
  477       VVector <$>
  478       case saw of
  479         NoReturnTrip _ -> V.replicateM (fromIntegral n) (ready <$> parseUninterpreted' saw ref app et)
  480         DoReturnTrip sym isVar st sc arg ->
  481           do
  482             elTy <- lift (termOfTValue sc et)
  483             V.generateM (fromIntegral n) (\i ->
  484               do
  485                 let newArg = ArgTermAt n elTy arg (fromIntegral i) 
  486                 el <- parseUninterpreted' (DoReturnTrip sym isVar st sc newArg) ref app et
  487                 pure (ready el)
  488               )
  489           
  490     VArrayType ity ety
  491       | Just (Some idx_repr) <- valueAsBaseType ity
  492       , Just (Some elm_repr) <- valueAsBaseType ety
  493       -> (VArray . SArray) <$>
  494           mkUninterpreted (BaseArrayRepr (Ctx.Empty Ctx.:> idx_repr) elm_repr) saw
  495 
  496     VDataType (ModuleIdentifier "Prelude.UnitType") [] []
  497       -> pure vUnit
  498 
  499     VDataType (ModuleIdentifier "Prelude.PairType") [TValue ty1, TValue ty2] []
  500       -> do x1 <- parseUninterpreted' (mapArgTerm ArgTermPairLeft saw) ref app ty1
  501             x2 <- parseUninterpreted' (mapArgTerm ArgTermPairRight saw) ref app ty2
  502             pure (vPair (ready x1) (ready x2))
  503 
  504     VDataType (ModuleIdentifier "Prelude.EmptyType") [] []
  505       -> pure vEmptyRecord
  506     VDataType (ModuleIdentifier "Prelude.RecordType")
  507       [VString fname, TValue ty1, TValue ty2] []
  508       -> do x1 <- parseUninterpreted' (mapArgTerm (`ArgTermRecordSelect` fname) saw) ref app ty1
  509             x2 <- parseUninterpreted' saw ref app ty2
  510             pure (vRecordValue (ready x1) (ready x2))
  511 
  512     _ -> fail $ "could not create uninterpreted symbol of type " ++ show ty
  513   where
  514 
  515   -- Get the next symbolic term of this type, and also record its interpretation
  516   -- back into SAW Core (if we need to do so).
  517   mkUninterpreted :: BaseTypeRepr t -> ReturnTrip sym -> StateT (UnintState sym) IO (SymExpr sym t)
  518   mkUninterpreted tyr argTerm =
  519     do
  520       mp <- get
  521       case MapF.lookup tyr mp of
  522 
  523         -- This happens when we are translating a symbolic variable with
  524         -- no arguments (see `parseUninterpretedTop`, first case)
  525         Nothing
  526           | DoReturnTrip sym True st sc arg <- saw ->
  527             lift (bindSAWTerm sym st tyr =<< reconstructArgTerm arg sc [])
  528             
  529         Just (Arr fn w (x : xs) atms) ->
  530           do
  531             let newTerm =
  532                   case argTerm of
  533                     NoReturnTrip _     -> []
  534                     DoReturnTrip _ _ _ _ t -> t : atms
  535             put (MapF.insert tyr (Arr fn w xs newTerm) mp)
  536             pure x
  537 
  538         _ -> panic "mkUninterpreted" ["Not enough uninterpreted results"]

  543 -- Symbolic function and its arguments.
  544 
  545 -- | A value of type @UnintApp f@ represents an uninterpreted function
  546 -- with the given 'String' name, applied to a list of argument values
  547 -- paired with a representation of their types. The context of
  548 -- argument types is existentially quantified.
  549 data UnintApp f =
  550   forall args. UnintApp String (Assignment f args) (Assignment BaseTypeRepr args)
  551 
  552 -- | Extract the string from an 'UnintApp'.
  553 stringOfUnintApp :: UnintApp f -> String
  554 stringOfUnintApp (UnintApp s _ _) = s
  555 
  556 -- | Make an 'UnintApp' with the given name and no arguments.
  557 mkUnintApp :: String -> UnintApp f
  558 mkUnintApp nm = UnintApp nm Ctx.empty Ctx.empty
  559 
  560 -- | Add a suffix to the function name of an 'UnintApp'.  This is used
  561 -- to modify the function name for different polymorphic instantiations,
  562 -- for example.
  563 suffixUnintApp :: String -> UnintApp f -> UnintApp f
  564 suffixUnintApp s (UnintApp nm args tys) = UnintApp (nm ++ s) args tys
  565 
  566 -- | Extend an 'UnintApp' with an additional argument.
  567 extendUnintApp :: UnintApp f -> f ty -> BaseTypeRepr ty -> UnintApp f
  568 extendUnintApp (UnintApp nm xs tys) x ty =
  569   UnintApp nm (Ctx.extend xs x) (Ctx.extend tys ty)

  571 -- | Flatten an 'SValue' to a sequence of components, each of which is
  572 -- a symbolic value of a base type (e.g. word or boolean), and add
  573 -- them to the list of arguments of the given 'UnintApp'. If the
  574 -- 'SValue' contains any values built from data constructors, then
  575 -- encode them as suffixes on the function name of the 'UnintApp'.
  576 applyUnintApp ::
  577   forall sym.
  578   (W.IsExprBuilder sym) =>
  579   sym ->
  580   UnintApp (SymExpr sym) ->
  581   SValue sym ->
  582   IO (UnintApp (SymExpr sym))
  583 applyUnintApp sym app0 v =
  584   case v of
  585     VVector xv                -> foldM (applyUnintApp sym) app0 =<< traverse force xv
  586     VBool sb                  -> return (extendUnintApp app0 sb BaseBoolRepr)
  587     VInt si                   -> return (extendUnintApp app0 si BaseIntegerRepr)
  588     VIntMod 0 si              -> return (extendUnintApp app0 si BaseIntegerRepr)
  589     VIntMod n si              -> do n' <- W.intLit sym (toInteger n)
  590                                     si' <- W.intMod sym si n'
  591                                     return (extendUnintApp app0 si' BaseIntegerRepr)
  592     VRational numer denom     -> do app1 <- applyUnintApp sym app0 (VInt numer)
  593                                     app2 <- applyUnintApp sym app1 (VInt denom)
  594                                     pure app2
  595     VWord (DBV sw)            -> return (extendUnintApp app0 sw (W.exprType sw))
  596     VArray (SArray sa)        -> return (extendUnintApp app0 sa (W.exprType sa))
  597     VWord ZBV                 -> return app0
  598     VCtorApp 0 _ []           -> return app0
  599     VCtorApp 0 _ [x, y]       -> do app1 <- applyUnintApp sym app0 =<< force x
  600                                     app2 <- applyUnintApp sym app1 =<< force y
  601                                     return app2
  602     VCtorApp n _ xs           -> foldM (applyUnintApp sym) app' =<< traverse force xs
  603                                    where app' = suffixUnintApp ("_" ++ show n) app0
  604     VNat n                    -> return (suffixUnintApp ("_" ++ show n) app0)
  605     VBVToNat w v'             -> applyUnintApp sym app' v'
  606                                    where app' = suffixUnintApp ("_" ++ show w) app0
  607     TValue (suffixTValue -> Just s)
  608                               -> return (suffixUnintApp s app0)
  609     VFun {} ->
  610       fail $
  611       "Cannot create uninterpreted higher-order function " ++
  612       show (stringOfUnintApp app0)
  613     _ ->
  614       fail $
  615       "Cannot create uninterpreted function " ++
  616       show (stringOfUnintApp app0) ++
  617       " with argument " ++ show v

  623 -- `ArgTerms` are used to remember the mappings between low-level symbolic
  624 -- terms and SAW core terms. 
  625 
  626 
  627 -- | An 'ArgTerm' is a description of how to reassemble a saw-core
  628 -- term from a list of the atomic components it is composed of.
  629 data ArgTerm
  630   = ArgTermVar
  631   | ArgTermBVZero -- ^ scBvNat 0 0
  632   | ArgTermToIntMod Natural ArgTerm -- ^ toIntMod n x
  633   | ArgTermFromIntMod Natural ArgTerm -- ^ fromIntMod n x
  634   | ArgTermRational ArgTerm ArgTerm -- ^ numerator, denominator
  635   | ArgTermVector Term [ArgTerm] -- ^ element type, elements
  636   | ArgTermUnit
  637   | ArgTermPair ArgTerm ArgTerm
  638   | ArgTermEmpty
  639   | ArgTermRecord FieldName ArgTerm ArgTerm
  640   | ArgTermRecordSelect ArgTerm FieldName
  641   | ArgTermConst Term
  642   | ArgTermApply ArgTerm ArgTerm
  643   | ArgTermAt Natural Term ArgTerm Natural
  644     -- ^ length, element type, list, index
  645   | ArgTermPairLeft ArgTerm
  646   | ArgTermPairRight ArgTerm
  647   | ArgTermBVToNat Natural ArgTerm
  648   | ArgTermErr String
  649     deriving Show
  650 
  651 -- | Reassemble a saw-core term from an 'ArgTerm' and a list of parts.
  652 -- The length of the list should be equal to the number of
  653 -- 'ArgTermVar' constructors in the 'ArgTerm'.
  654 reconstructArgTerm :: ArgTerm -> SharedContext -> [Term] -> IO Term
  655 reconstructArgTerm atrm sc ts =
  656   do (t, unused) <- parse atrm ts
  657      unless (null unused) $ fail "reconstructArgTerm: too many function arguments"
  658      return t
  659   where
  660     parse :: ArgTerm -> [Term] -> IO (Term, [Term])
  661     parse at ts0 =
  662       case at of
  663         ArgTermVar ->
  664           case ts0 of
  665             x : ts1 -> return (x, ts1)
  666             [] -> fail "reconstructArgTerm: too few function arguments"
  667         ArgTermBVZero ->
  668           do z <- scNat sc 0
  669              x <- scBvNat sc z z
  670              return (x, ts0)
  671         ArgTermToIntMod n at1 ->
  672           do n' <- scNat sc n
  673              (x1, ts1) <- parse at1 ts0
  674              x <- scToIntMod sc n' x1
  675              pure (x, ts1)
  676         ArgTermFromIntMod n at1 ->
  677           do n' <- scNat sc n
  678              (x1, ts1) <- parse at1 ts0
  679              x <- scFromIntMod sc n' x1
  680              pure (x, ts1)
  681         ArgTermRational numer denom ->
  682           do (numer', ts1) <- parse numer ts0
  683              (denom', ts2) <- parse denom ts1
  684              rat <- scRational sc numer' denom'
  685              pure (rat, ts2)
  686         ArgTermVector ty ats ->
  687           do (xs, ts1) <- parseList ats ts0
  688              x <- scVectorReduced sc ty xs
  689              return (x, ts1)
  690         ArgTermUnit ->
  691           do x <- scUnitValue sc
  692              return (x, ts0)
  693         ArgTermPair at1 at2 ->
  694           do (x1, ts1) <- parse at1 ts0
  695              (x2, ts2) <- parse at2 ts1
  696              x <- scPairValue sc x1 x2
  697              return (x, ts2)
  698         ArgTermEmpty ->
  699           do x <- scRecordValue sc []
  700              pure (x, ts0)
  701         ArgTermRecord fname at1 at2 ->
  702           do s <- scString sc fname
  703              (x1, ts1) <- parse at1 ts0
  704              (x2, ts2) <- parse at2 ts1
  705              ty1 <- scTypeOf sc x1
  706              ty2 <- scTypeOf sc x2
  707              x <- scGlobalApply sc "Prelude.RecordValue" [s, ty1, ty2, x1, x2]
  708              pure (x, ts2)
  709         ArgTermRecordSelect at1 fname ->
  710           do (x1, ts1) <- parse at1 ts0
  711              x <- scRecordSelect sc x1 fname
  712              pure (x, ts1)
  713         ArgTermConst x ->
  714           do return (x, ts0)
  715         ArgTermApply at1 at2 ->
  716           do (x1, ts1) <- parse at1 ts0
  717              (x2, ts2) <- parse at2 ts1
  718              x <- scApply sc x1 x2
  719              return (x, ts2)
  720         ArgTermAt n ty at1 i ->
  721           do n' <- scNat sc n
  722              (x1, ts1) <- parse at1 ts0
  723              i' <- scNat sc i
  724              x <- scAt sc n' ty x1 i'
  725              return (x, ts1)
  726         ArgTermPairLeft at1 ->
  727           do (x1, ts1) <- parse at1 ts0
  728              x <- scPairLeft sc x1
  729              return (x, ts1)
  730         ArgTermPairRight at1 ->
  731           do (x1, ts1) <- parse at1 ts0
  732              x <- scPairRight sc x1
  733              return (x, ts1)
  734         ArgTermBVToNat w at1 ->
  735           do (x1, ts1) <- parse at1 ts0
  736              x <- scBvToNat sc w x1
  737              pure (x, ts1)
  738         ArgTermErr msg -> fail msg
  739 
  740     parseList :: [ArgTerm] -> [Term] -> IO ([Term], [Term])
  741     parseList [] ts0 = return ([], ts0)
  742     parseList (at : ats) ts0 =
  743       do (x, ts1) <- parse at ts0
  744          (xs, ts2) <- parseList ats ts1
  745          return (x : xs, ts2)
  746 
  747 -- | Given a type and value encoded as 'SValue's, construct an
  748 -- 'ArgTerm' that builds a term of that type from local variables with
  749 -- base types. The number of 'ArgTermVar' constructors should match
  750 -- the number of arguments appended by 'applyUnintApp'.
  751 mkArgTerm :: forall sym. SharedContext -> TValue (What4 sym) -> SValue sym -> IO ArgTerm
  752 mkArgTerm sc ty val =
  753   case (ty, val) of
  754     (VBoolType, VBool _) -> return ArgTermVar
  755     (VIntType, VInt _)   -> return ArgTermVar
  756     (_, VWord ZBV)       -> return ArgTermBVZero     -- 0-width bitvector is a constant
  757     (_, VWord (DBV _))   -> return ArgTermVar
  758     (_, VArray{})        -> return ArgTermVar
  759     (VIntModType n, VIntMod _ _) -> pure (ArgTermToIntMod n ArgTermVar)
  760     (VRationalType, VRational numer denom) ->
  761       do numer' <- mkArgTerm @sym sc VIntType (VInt numer)
  762          denom' <- mkArgTerm @sym sc VIntType (VInt denom)
  763          pure (ArgTermRational numer' denom')
  764 
  765     (VVecType _ ety, VVector vv) ->
  766       do vs <- traverse force (V.toList vv)
  767          xs <- traverse (mkArgTerm sc ety) vs
  768          ety' <- termOfTValue sc ety
  769          return (ArgTermVector ety' xs)
  770 
  771     (VDataType (ModuleIdentifier "Prelude.UnitType") [] [],
  772      VCtorApp 0 _ [])
  773                          -> return ArgTermUnit
  774     (VDataType (ModuleIdentifier "Prelude.PairType") [TValue ty1, TValue ty2] [],
  775      VCtorApp 0 _ [v1, v2]) ->
  776       do x1 <- mkArgTerm sc ty1 =<< force v1
  777          x2 <- mkArgTerm sc ty2 =<< force v2
  778          return (ArgTermPair x1 x2)
  779 
  780     (VDataType (ModuleIdentifier "Prelude.EmptyType") [] [],
  781      VCtorApp 0 _ []) ->
  782       pure ArgTermEmpty
  783     (VDataType _nm [VString fname, TValue ty1, TValue ty2] [],
  784      VCtorApp 0 _ [v1, v2]) ->
  785       do x1 <- mkArgTerm sc ty1 =<< force v1
  786          x2 <- mkArgTerm sc ty2 =<< force v2
  787          pure (ArgTermRecord fname x1 x2)
  788 
  789     (VDataType nmi ps _, VCtorApp n _ vv) ->
  790       do mvi <- scResolveQualName sc (toQualName nmi)
  791          vi <-
  792            case mvi of
  793              Just vi -> pure vi
  794              Nothing -> panic "mkArgTerm" ["Data type name not found: " <> toAbsoluteName nmi]
  795          mm <- scGetModuleMap sc
  796          dt <-
  797            case lookupVarIndexInMap vi mm of
  798              Just (ResolvedDataType dt) -> pure dt
  799              _ -> panic "mkArgTerm" ["Data type not found: " <> toAbsoluteName nmi]
  800          let ctor = dtCtors dt !! n
  801          ps' <- traverse (termOfSValue sc) ps
  802          vv' <- traverse (termOfSValue sc <=< force) vv
  803          x   <- scConstApply sc (ctorName ctor) (ps' ++ vv')
  804          return (ArgTermConst x)
  805 
  806     (_, TValue tval) ->
  807       do x <- termOfTValue sc tval
  808          pure (ArgTermConst x)
  809 
  810     (_, VNat n) ->
  811       do x <- scNat sc n
  812          pure (ArgTermConst x)
  813 
  814     (_, VBVToNat w v) ->
  815       do let w' = fromIntegral w -- FIXME: make w :: Natural to avoid fromIntegral
  816          x <- mkArgTerm sc (VVecType w' VBoolType) v
  817          pure (ArgTermBVToNat w' x)
  818 
  819     _ -> fail $ "could not create uninterpreted function argument of type " ++ show ty

  824 -- Uninterpreted function cache
  825 
  826 type SymFnCache sym =
  827   Map W.SolverSymbol (MapF (Assignment BaseTypeRepr) (SymFnWrapper sym))
  828 
  829 lookupSymFn ::
  830   W.SolverSymbol -> Assignment BaseTypeRepr args -> BaseTypeRepr ty ->
  831   SymFnCache sym -> Maybe (W.SymFn sym args ty)
  832 lookupSymFn s args ty cache =
  833   do m <- Map.lookup s cache
  834      SymFnWrapper fn <- MapF.lookup (Ctx.extend args ty) m
  835      return fn
  836 
  837 insertSymFn ::
  838   W.SolverSymbol -> Assignment BaseTypeRepr args -> BaseTypeRepr ty ->
  839   W.SymFn sym args ty -> SymFnCache sym -> SymFnCache sym
  840 insertSymFn s args ty fn = Map.alter upd s
  841   where
  842     upd Nothing = Just (MapF.singleton (Ctx.extend args ty) (SymFnWrapper fn))
  843     upd (Just m) = Just (MapF.insert (Ctx.extend args ty) (SymFnWrapper fn) m)
  844 
  845 mkSymFn ::
  846   forall sym args ret. (IsSymExprBuilder sym) =>
  847   sym -> IORef (SymFnCache sym) ->
  848   String -> Assignment BaseTypeRepr args -> BaseTypeRepr ret ->
  849   IO (W.SymFn sym args ret)
  850 mkSymFn sym ref nm args ret =
  851   do let s = W.safeSymbol nm
  852      cache <- readIORef ref
  853      case lookupSymFn s args ret cache of
  854        Just fn -> pure fn
  855        Nothing ->
  856          do fn <- W.freshTotalUninterpFn sym s args ret
  857             writeIORef ref (insertSymFn s args ret fn cache)
  858             pure fn
